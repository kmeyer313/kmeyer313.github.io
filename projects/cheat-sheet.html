Selectors

SELECTOR MEANING CSS EXAMPLE HTML SYNTAX
UNIVERSAL Applies to all elements in the document * {} n/a
TYPE Matches element names h1, h2, h3 {} <h1>, <h2>, <h3>
CLASS Matches an element whose class attribute has a value that matches the one specified after the period (or full stop) symbol .value {} OR p.value {} <section class="value">
ID Matches an element whose id attribute has a value that matches the one specified after the pound or hash symbol #value {} <section id="value">
CHILD Matches an element that is a direct child of another li>a {} <h1>, <h2>, <h3>
DESCENDANT Matches an element that is a descendent of another specified element (not just a direct child of that element) p a {} <p><a></a></p>
ADJACENT SIBLING Matches an element that is the next sibling of another h1+p
GENERAL SIBLING Matches an element that is a sibling of another, although it does not have to be the directly preceding element h1~p
The information from this table is from HTML & CSS Design and Build Websites by Jon Duckett
